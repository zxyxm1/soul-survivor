# Soul Survivor: 地下生存 — 游戏玩法介绍文档

> **版本:** 1.0  
> **类型:** Roguelike / 生存割草 / 命令行小游戏  
> **风格:** 吸血鬼幸存者 × UNDERTALE  
> **平台:** Windows / Linux / macOS 命令行  
> **课程:** 程序设计基础 A - 课程设计作品

---

## 一、游戏概述

**Soul Survivor: 地下生存** 是一款运行在命令行终端中的 Roguelike 生存割草游戏。你扮演一个坠入地下世界的灵魂，面对源源不断涌来的怪物群，依靠自动攻击武器和走位策略存活下去。

> *"在这地下世界，你的灵魂仍闪耀着光芒。保持决心，存活下去..."*

游戏融合了《吸血鬼幸存者》的自动战斗+割草升级玩法与《UNDERTALE》的怪物设计/决心主题，在 60×17 的 ASCII 竞技场中展开快节奏的弹幕躲避战。

## 二、核心玩法

### 2.1 基本规则

| 要素 | 说明 |
|------|------|
| **目标** | 在无限波次中尽可能存活更久，击杀更多怪物 |
| **失败条件** | HP 降至 0 且无「决心」复活次数 |
| **胜利条件** | 无（无限生存挑战，追求高分） |
| **视角** | 俯视固定竞技场（60×17 格） |
| **操作** | 实时控制（非回合制） |

### 2.2 操作方式

| 按键 | 功能 |
|------|------|
| **W A S D** | 上/左/下/右 移动（支持斜向移动） |
| **空格** | 闪避冲刺（向远离最近敌人方向冲刺3格，路径伤害 + 短暂无敌） |
| **P** | 暂停 / 继续 |
| **Q / ESC** | 返回标题菜单 |

### 2.3 战斗机制

#### 自动攻击系统
玩家最多持有 **4 把武器**，每把武器独立冷却，自动锁定范围内的敌人：

- **近战武器**：瞬间造成伤害 + 显示劈砍动画
- **远程武器**：发射弹幕（飞行道具），命中后造成伤害

#### 武器一览

| 武器 | 类型 | 伤害 | 冷却(帧) | 范围(格) | 特殊效果 |
|------|------|------|----------|----------|----------|
| 🏒 **树枝 (Stick)** | 近战 | +3 | 48 | 3 | 基础武器，同时攻击1个目标 |
| 🦴 **骨头攻击 (Bone)** | 远程 | +5 | 72 | 5 | **穿透**：弹幕命中后继续飞行，攻击3个目标 |
| 🔥 **火焰魔法 (Fire)** | 近战 | +4 | 88 | 2 | **范围灼烧 (AoE)**：半径2格溅射伤害 |
| 🔪 **小刀 (Knife)** | 近战 | +2 | 24 | 2 | 攻速极快（冷却最短） |
| ⚡ **闪电 (Lightning)** | 远程 | +3 | 60 | 3 | **连锁**：同时攻击4个目标 |

#### 伤害计算

```
伤害 = max(攻击力 - 敌人防御力, 1)
```

- 玩家受伤后获得 **1秒无敌时间**（60帧）
- 开局有 **约3.7秒无敌时间**（220帧）

### 2.4 特殊技能 — 闪避

| 属性 | 数值 |
|------|------|
| 冷却时间 | 440帧（约7.3秒 @ 60FPS） |
| 冲刺距离 | 3格 |
| 冲刺方向 | 远离最近敌人的方向 |
| 效果 | 路径上敌人受到 **攻击力×2** 伤害 + 短暂无敌（6帧） |

## 三、敌人系统

### 3.1 敌人一览

参考 UNDERTALE 怪物设计，共 7 种敌人：

| 敌人 | 外观 | HP | ATK | DEF | 速度(帧) | EXP | 特点 |
|------|------|-----|-----|-----|----------|-----|------|
| 🐸 **Froggit** | ♟ | 8 | 2 | 0 | 36 | 5 | 基础小怪，移动慢 |
| 🦋 **Whimsun** | ♤ | 5 | 1 | 0 | 28 | 3 | 最弱但移动快 |
| 🟡 **Moldsmal** | ◉ | 12 | 1 | 2 | 60 | 6 | 史莱姆型，高防御低速 |
| 🐰 **Loox** | ♢ | 10 | 3 | 0 | 36 | 7 | 高攻击普通怪 |
| 🌿 **Vegetoid** | ♧ | 15 | 2 | 1 | 44 | 8 | 植物型，血量较高 |
| 👾 **Migosp** | ♠ | 25 | 4 | 1 | 28 | 15 | 小Boss级别，高攻高速 |
| 🐕 **Greater Dog** | ♛ | 60 | 6 | 3 | 44 | 50 | **Boss**，每5波出现 |

### 3.2 敌人 AI

1. **追踪玩家**：敌人每 N 帧（由速度决定）朝玩家方向移动1格
2. **避免堆叠**：80%概率沿主轴移动，20%随机偏移，且不与其他敌人重叠
3. **碰撞伤害**：敌人与玩家重合时造成伤害，并将敌人弹开

### 3.3 波次系统

| 波次 | 规则 |
|------|------|
| Wave 1-2 | 只出现前3种基础敌人 (Froggit, Whimsun, Moldsmal) |
| Wave 3-4 | 可生成前6种敌人 (不包括Boss) |
| Wave 5+ | Boss 波（每5波生成一个 Greater Dog） |
| Wave 6+ | 全部种类随机混合 |

**难度递增**：每波敌人属性 **+25%**（乘法叠加），第5波后移速加快，第3波后防御递增。

## 四、升级系统

### 4.1 经验与升级

- 击杀敌人获得经验（EXP），受 `经验加成` 影响
- 经验条满后进入 **升级画面**，从 3 个随机选项中选 1 个
- 升级所需经验：`(当前等级 + 1) × 15`

### 4.2 升级选项

| 类别 | 选项 | 效果 |
|------|------|------|
| 💪 属性 | **攻击力+3** | 提升基础攻击力 |
| 🏃 属性 | **速度+1** | 减少移动间隔 |
| ❤️ 属性 | **最大HP+8** | 提升生命上限并治疗 |
| 🛡️ 属性 | **防御力+2** | 减少受到的伤害 |
| 💚 恢复 | **治疗** | 恢复10点HP（仅HP不满时出现） |
| ⚔️ 武器 | **骨头/火焰/小刀/闪电** | 获得新武器（最多4把，不重复） |
| 📏 范围 | **攻击范围+1** | 所有武器范围+1 |
| ✨ 经验 | **经验加成** | 经验获取+25% |
| 🧲 磁铁 | **磁铁效果** | 扩大经验自动拾取范围 |
| 💎 特殊 | **决心+1** | 死亡时自动复活（最多持有2次） |

## 五、特殊机制

### 5.1 决心 (Determination)

> *源于 UNDERTALE 的核心概念*

| 属性 | 说明 |
|------|------|
| 初始值 | 0 |
| 最大层数 | 2 |
| 效果 | HP归零时自动复活，恢复 **50% 最大生命** + 2秒无敌 |
| 获取 | 升级时选择「决心+1」选项 |

### 5.2 磁铁效果

- 初始经验拾取范围：3格（曼哈顿距离）
- 选「磁铁效果」每次 +2 格
- 范围内已死亡但未拾取的经验自动收集

### 5.3 经验加成

- 初始 100%（即 1x）
- 每次选「经验加成」+25%
- 实际获取经验 = 基础经验 × (经验加成 / 100)

## 八、存档系统

- **存档文件**: `soul_survivor_save.dat`（二进制格式）
- **记录内容**: 最高分、总击杀数、总游戏次数、玩家名称
- **自动保存**: 每次游戏结束时更新
- 标题画面选 `[2]` 查看记录

## 九、小贴士

1. **走位是核心**：武器自动攻击，你要做的就是走位躲避！
2. **尽早拿第二把武器**：多样化的武器组合大幅提升清怪效率
3. **速度很关键**：后期敌人越来越快，速度不够会被围死
4. **决心 = 第二条命**：最多存2次，关键时刻救命
5. **火焰 + 骨头 是强力组合**：AoE 清杂 + 穿透打 Boss
6. **善用闪避**：被包围时空格冲刺可以穿出去，还能造成伤害
7. **经验磁铁**：后期怪物多时，扩大拾取范围避免浪费经验
