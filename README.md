# Soul Survivor: 地下生存

> 🎮 吸血鬼幸存者 × UNDERTALE 风格命令行 Roguelike 游戏  
> 📟 在终端里割草，用决心活下去  
> 🎓 程序设计基础 A — 课程设计作品

---

## 📖 简介

坠入地下世界的你，灵魂仍在闪耀。面对源源不断的怪物浪潮，拿起武器、走位闪避、获取升级，在 ASCII 竞技场中存活尽可能久。

游戏融合了 **《吸血鬼幸存者》** 的自动战斗 + 割草升级玩法与 **《UNDERTALE》** 的怪物设计与决心主题，全部运行在命令行终端中。

---

## 🎬 快速开始

### 编译

```bash
# Windows (MinGW / MSVC)
gcc -o soul_survivor.exe main.c player.c enemy.c combat.c render.c effect.c data.c -lm

# Linux / macOS
gcc -o soul_survivor main.c player.c enemy.c combat.c render.c effect.c data.c -lm
```

### 运行

```bash
./soul_survivor   # Linux/macOS
soul_survivor.exe # Windows
```

> ⚠️ Windows 用户请使用 **Windows Terminal** 或支持 ANSI 转义序列的终端以获得最佳体验。

## 🎮 操作方式

| 按键 | 功能 |
|------|------|
| **W A S D** | 上下左右移动（支持斜向移动） |
| **空格** | 闪避冲刺（远离敌人 + 路径伤害 + 短暂无敌） |
| **P** | 暂停 / 继续 |
| **Q / ESC** | 返回标题菜单 |

| 武器 | 类型 | 伤害 | 特殊效果 |
|------|------|------|----------|
| 树枝 | 近战 | +3 | 基础武器，入门之选 |
| 骨头攻击 | 远程 | +5 | 穿透弹幕，命中后继续飞行 |

## ⬆️ 升级系统

击杀敌人获取经验 → 升级 → 三选一：

| 升级 | 效果 |
|------|------|
| 攻击力+3 | 提升基础伤害 |
| 速度+1 | 移动更灵活 |
| 最大HP+8 | 更多血量 |
| 防御力+2 | 减少伤害 |
| 治疗 | 恢复10HP |
| 新武器 | 获得未持有的武器 |
| 攻击范围+1 | 所有武器范围扩大 |
| 经验加成 | 经验获取+25% |
| 磁铁效果 | 扩大经验拾取范围 |
| 决心+1 | 死亡时复活一次 |

## 💎 核心机制

- **自动战斗** — 武器自动索敌攻击，你只需专注走位
- **决心复活** — Inherited from UNDERTALE，HP 归零时消耗决心层数复活
- **闪避冲刺** — 紧急脱困技能，沿远离敌人方向冲刺并对路径造成伤害
- **波次生存** — 无限波次，挑战最高分
- **本地存档** — 二进制文件保存最高分记录

```

```
